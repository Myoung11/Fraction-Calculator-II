# Fraction-Calculator-II
# C ++
Building a class for Fraction ADT II

Add, Subtract, Multiply, Divide, Reduce, Convert to decimals, Compare two fractions.
# Example Input/Output
Program output:

 7/3 + 1/3 = 8/3
 7/3 - 1/3 = 2 or 2/1
 7/3 * 1/3 = 7/9
 7/3 / 1/3 = 7 or 7/1
 7/3 is:
 > 1/3 according to the overloaded > operator
 >= 1/3 according to the overloaded < operator
